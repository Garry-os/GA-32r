# GA-32
## Overview
The official name is GA-32, the repository "GA-32r" is just a rewrite of the architecture in **Rust**.  
This is no mean a production-ready CPU architecture. I have written this for fun & educational purposes.  
This repository includes the assembler, emulator, and softwares.

### Architecture
The **GA-32** architecture is a **RISC-like** architecture with fixed instruction width.  
You can read more in the CPU specification.

The **Specifications** are in the `specs/` folder.
